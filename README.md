# 🚕 Uber Trip Analytics Project

## 📌 Project Overview

This project analyzes Uber trip data using SQL, Python, and Power BI to uncover revenue trends, trip demand patterns, and customer behavior insights.

The project follows a complete data analytics workflow:

* Data Cleaning & Preparation
* SQL Analysis
* Python Exploratory Data Analysis (EDA)
* Interactive Power BI Dashboard
* Business Insights & Recommendations

---

## 🛠️ Tools & Technologies

* SQL (MySQL)
* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
* Power BI

---

## 📂 Dataset Information

The dataset contains approximately 194,000+ Uber trips and includes:

* Ride ID
* Pickup Date & Time
* Fare Amount
* Passenger Count
* Pickup Coordinates
* Ride Hour
* Ride Month
* Day Name
* Time Category

---

## 💡 Key Business Insights

* Revenue peaked in May.
* Evening hours generated the highest trip demand.
* Friday produced the highest revenue.
* Single-passenger trips dominated ride demand.
* Revenue is concentrated during peak commuting periods.
